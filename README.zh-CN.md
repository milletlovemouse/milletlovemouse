## 项目亮点

- `rtc-chatroom`：基于 Vue 3、TypeScript、Vite、WebRTC 和 Socket.IO 的多人视频聊天室。
- `chatroom`：基于 React 和 TypeScript 的实时音视频聊天室实现，覆盖 WebRTC 信令流程。
- `chatroom-server`：面向前端练习项目的轻量 Node.js / Express / Socket.IO 服务。
- `rtc-client`：基于浏览器 WebRTC API 封装的视频通话 SDK，已发布为 `@houwx/rtc-client`。
